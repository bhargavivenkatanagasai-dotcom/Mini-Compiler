# 🚀 Mini Compiler - Lexical Analyzer

A simple **Mini Compiler (Lexical Analyzer)** built using **Python (Flask)**, **HTML, CSS, and JavaScript**.  
It performs lexical analysis on C-like code and displays tokens in a structured, color-coded UI.

---

## ✨ Features

- Keyword detection  
- Identifier detection  
- Number recognition  
- Operator detection  
- Symbol detection  
- Token counting  
- Color-coded output (IDE-style UI)  
- Real-time analysis using Flask API  
- Clean and modern interface  

---

## 🛠️ Tech Stack

- Python (Flask)
- HTML5
- CSS3
- JavaScript (Fetch API)

## 📊 Output Example

Token Type   | Value  
------------ | ------

---

## 🔄 Working Flow

1. User enters code in editor  
2. Frontend sends code to Flask backend  
3. Backend performs lexical analysis  
4. Tokens are generated  
5. Frontend displays structured table output
